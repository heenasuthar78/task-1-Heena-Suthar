# Rule-Based AI Chatbot 🤖

## Project Overview
This is a simple Rule-Based AI Chatbot built using Python. It responds to predefined user inputs using if-else statements and runs continuously until the user exits the program.

## Features
- Greets the user
- Responds to simple questions
- Uses if-else logic
- Runs in a continuous loop
- Exits when the user types "bye"

## Technologies Used
- Python 3

## How to Run
1. Install Python 3.
2. Download or clone this repository.
3. Open the project folder.
4. Run:
   ```bash
   python chatbot.py
   ```

## Example

```
You: hi
Bot: Hello! How can I help you?

You: how are you
Bot: I am fine. Thank you!

You: bye
Bot: Goodbye!
```
